# Recursos para aprender JavaScript

## 1. [JavaScript Info](https://javascript.info/)
Un sitio web completo para aprender JavaScript desde lo más básico hasta conceptos avanzados. Contiene tutoriales interactivos, ejemplos y explicaciones claras de todos los aspectos del lenguaje.

---

## 2. [Eloquent JavaScript](https://eloquentjavascript.net/)
Libro en línea gratuito que cubre JavaScript de manera profunda y práctica. Incluye ejercicios y proyectos que ayudan a entender tanto la sintaxis como las buenas prácticas del lenguaje.

---

## 3. [You Don’t Know JS (YDKJS)](https://github.com/getify/You-Dont-Know-JS?tab=readme-ov-file)
Serie de libros en GitHub que exploran JavaScript a un nivel muy profundo. Ideal para desarrolladores que ya tienen conocimientos básicos y quieren dominar el lenguaje a fondo.

---

## 4. [100 Ejercicios de Funciones en JS](https://www.100jsfunctions.com/exercises)
Colección de 100 ejercicios prácticos enfocados en funciones de JavaScript. Perfecto para practicar y reforzar conceptos de programación funcional y manipulación de arrays y strings.
